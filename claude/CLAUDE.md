# User rules

## Output

Keep output clear and concise. Avoid long explanations unless asked for.

## Worktrees

Worktree directories should be siblings of the main checkout (`../<main-checkout>-<branch-slug>`). e.g. a `feat/lsp-folding` branch of a `neovim/` checkout goes in `../neovim-feat-lsp-folding/`, and a `PROJ-123/fix-crash` branch goes in `../neovim-proj-123-fix-crash/`.
